# 🧙‍♂️ Harry Potter Angular App (COMP3133 Lab Test)

This project is an Angular-based web application that displays data from the Harry Potter universe, including characters, spells, books, and movies. The application fetches data from an external API and presents it in a structured UI.

---

## 🚀 Live Demo

Deployed on Vercel:  
👉 https://harry-potter-lab-test.vercel.app

---

## 🛠️ Technologies Used

- Angular
- TypeScript
- HTML / CSS
- Vercel (Deployment)
- External REST API

---

## 📦 Features

- View Harry Potter characters
- View spells with descriptions
- View books and related information
- View movies from the wizarding world
- Dynamic data fetching from API
- Clean and responsive UI

---

## 🔗 API Integration

This project uses a public Harry Potter API to fetch data.

Example endpoints used:
- `/characters`
- `/spells`
- `/books`
- `/movies`

Data is fetched using Angular's `HttpClient` and displayed using components and templates.
